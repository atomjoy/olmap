# Map marker with popup using OpenLayers

How to create a simple map with a marker using OpenLayers maps.

## Run

```sh
cd olmap
npm install
npm start
```

## Build for production

```sh
npm run build
```

## To get started

<https://openlayers.org/doc/quickstart.html>
